# Smart-Resume-Analyzer

1. Project Overview

Smart Resume Analyzer is a web-based application that analyzes a user's resume and compares it with a selected job role. The system evaluates the resume content, identifies relevant skills, and provides suggestions to improve the resume. This helps job seekers understand how well their resume matches industry requirements.


---

2. Features

User login and authentication

Resume upload (PDF / DOCX)

Job role selection

Resume skill extraction

Resume score calculation

Identification of missing skills

Suggestions for resume improvement

Dashboard to view results



---

3. How It Works

1. The user logs into the system.


2. The user uploads their resume.


3. The user selects a job role.


4. The system extracts text and skills from the resume.


5. The extracted skills are compared with job role requirements.


6. The system generates a resume score and provides suggestions for improvement.

5. System Architecture

The system follows a client-server architecture.
The user interacts with the frontend interface to upload the resume and select a job role. The backend processes the resume using text analysis techniques and compares the extracted skills with job role requirements. The results are then displayed to the user through the dashboard.
